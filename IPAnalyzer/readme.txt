Данное консольное приложение анализирует журнал с IP-адресами и временными метками доступа.
Оно фильтрует записи по диапазону IP-адресов и указанному интервалу времени, после чего сохраняет результаты в файл.

Входные данные:
Журнал должен содержать строки формата:

<IP-адрес>:<Дата и время доступа>

Пример:
192.168.1.15:2024-10-22 14:23:45
10.0.0.5:2024-10-22 15:00:00
192.168.1.20:2024-10-23 16:45:12

Параметры запуска:
Приложение поддерживает несколько способов задания параметров:

Аргументы командной строки — можно передать параметры прямо при запуске программы.
Конфигурационный файл (config.json) — можно указать параметры в JSON-файле.
Значения по умолчанию — используются, если ни аргументы, ни файл не заданы.
Формат команды:

<название_программы> [logFilePath] [outputFilePath] [ipRangeStart] [ipRangeEnd] [startTime] [endTime]

Пример команды:

MyApp.exe log.txt output.txt 192.168.1.10 192.168.1.100 "2024-10-22 00:00:00" "2024-10-23 23:59:59"

Параметры:

logFilePath — путь к файлу журнала (по умолчанию "log.txt").
outputFilePath — путь к выходному файлу (по умолчанию "output.txt").
ipRangeStart — начальный IP-адрес диапазона (по умолчанию "192.168.1.10").
ipRangeEnd — конечный IP-адрес диапазона (по умолчанию "192.168.1.100").
startTime — начало временного диапазона в формате "yyyy-MM-dd HH:mm" (по умолчанию "2024-10-22 00:00:00").
endTime — конец временного диапазона в формате "yyyy-MM-dd HH:mm" (по умолчанию "2024-10-23 23:59:59").

Использование конфигурационного файла:

Приложение поддерживает загрузку параметров из файла config.json.
Файл должен быть размещён в одной директории с программой и иметь следующий формат:

{
  "LOG_FILE_PATH": "log.txt",
  "OUTPUT_FILE_PATH": "output.txt",
  "IP_RANGE_START": "192.168.1.10",
  "IP_RANGE_END": "192.168.1.100",
  "START_TIME": "2024-10-22 00:00:00",
  "END_TIME": "2024-10-23 23:59:59"
}

Порядок приоритета параметров:
Аргументы командной строки.
Конфигурационный файл (config.json).
Значения по умолчанию.
Результат работы:
Программа создаст выходной файл, где будут перечислены IP-адреса и количество обращений за указанный интервал времени, например:

192.168.1.15 - 3 обращений
192.168.1.20 - 1 обращение
