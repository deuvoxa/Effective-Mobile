DeliveryService API

Описание:
API для службы доставки, которое фильтрует заказы в зависимости
от количества обращений в конкретном районе города и времени обращения с и по

Настройка и конфигурирование:
1. Склонируйте репозиторий
2. Соберите и запустите проект

API Методы:

POST /api/orders/load

Загружает заказы из текстового файла. Формат данных:
вес (в кг);район;дата и время доставки
Пример строки: 2.5;Центр;2024-10-23 10:30:00

Файл для примера: orders.txt

GET /api/orders/filter

Фильтрует заказы по району и интервалу времени

Параметры:
- area (string): Название района
- startTime (DateTime): Время начала фильтрации
- endTime (DateTime): Время окончания фильтрации

Тестирование:
Самый удобный способ протестировать API — использовать Swagger UI, доступный после запуска приложения
